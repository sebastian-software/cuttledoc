# Test Fixtures

Audio files for testing speech recognition backends.

## Files

| File            | Language | Duration | Description                             |
| --------------- | -------- | -------- | --------------------------------------- |
| `beamen-de.ogg` | German   | ~5 min   | Philosophical essay about teleportation |
| `beamen-en.ogg` | English  | ~5 min   | Same essay, English version             |

The `.md` files contain the original text transcripts.

## Origin & License

- **Text**: Generated by OpenAI ChatGPT (imaginative essays)
- **Audio**: Synthesized using ElevenLabs TTS

Both text and audio were generated by the project maintainers and are released under the same license as the project itself (MIT).

## Usage

These fixtures are used for integration testing of speech recognition accuracy. The markdown files serve as ground truth for comparing transcription output.
